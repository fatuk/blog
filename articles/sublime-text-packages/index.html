Пакеты для Sublime Text 2

03 ноя 2014

Написал Андрей Фатюк

sublime textpackagesпакетыpackage control

Список пакетов, которыми я пользуюсь. Буду его периодически обновлять и ставить дату обновления.

  • AutoFileName — автодополнение путей к файлам.
  • Backbone.js — добавляет набор сниппетов для стандартных структур Backbone — view, collection, route, model, etc. Лично мне сильно экономит время.
  • CSS Format — форматирует стили CSS/SASS/SCSS/LESS.
  • Emmet — как по мне, один из основных плагинов для верстки.
  • FileDiffs — создает diff между двумя файлами, может сравнивать файл с буфером обмена.
  • GitGutter — помечает новые, изменившиеся или удаленные строки в файле.
  • JavaScript Snippets — оптимальный набор сниппетов для JS.
  • JavaScript Beautify — форматирование JS.
  • LESS — подсветка синтаксиса less.
  • Markdown Extended
  • SublimeLinter — проверка синтаксиса.
  • WebFont — быстрое подключение шрифтов, можно скачать.
  • cdnjs — подключение библиотек из cdn облака cdnjs.com
